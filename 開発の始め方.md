# AimZ 開発の始め方

## 1. 初回セットアップ（最初の1回だけ）

ターミナルを開いて以下を順番に実行してください。

```bash
# このフォルダに移動
cd ~/Documents/App/aimz

# 必要なパッケージをインストール（少し時間がかかります）
npm install
```

## 2. Appwrite のセットアップ

1. https://cloud.appwrite.io にアクセスしてアカウントを作成
2. 新しいプロジェクトを作成（プロジェクト名: AimZ）
3. `.env.example` をコピーして `.env` という名前で保存
4. `.env` にプロジェクト ID などを入力

```bash
# .env.example をコピー
cp .env.example .env

# .env をテキストエディタで開いて編集
open .env
```

## 3. ローカルで確認する

```bash
cd ~/Documents/App/aimz
npm start
```

ブラウザが自動で開き `http://localhost:3000` が表示されます。

## 4. GitHub にコードを保存する

```bash
# GitHubでリポジトリを作成したら実行
git init
git add .
git commit -m "first commit"
git branch -M main
git remote add origin https://github.com/あなたのユーザー名/aimz.git
git push -u origin main
```

## 5. 本番公開（GitHub Pages）

```bash
# package.json の homepage を自分の URL に変更してから実行
npm run deploy
```

---

## よく使うコマンド

| コマンド | 内容 |
|---------|------|
| `npm start` | ローカルで確認 |
| `npm run deploy` | 本番公開 |
| `git add . && git commit -m "変更内容" && git push` | コード保存 |
